Tools

HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It tells the browser how to display text, images, links, and other elements on a webpage.

CSS

CSS stands for Cascading Style Sheets. It controls the appearance and layout of HTML elements. While HTML is for structure, CSS is for styling — like colors, fonts, spacing, and responsiveness.

JavaScript

JavaScript is a programming language that is primarily used to create interactive and dynamic content on websites. It runs in the browser and works alongside HTML and CSS to build modern web applications.

Bootstrap

Bootstrap is a free and open-source CSS framework used to create responsive, mobile-first websites quickly and easily. It provides pre-designed components, like buttons, navbars, modals.

Tailwind

Tailwind CSS is a utility-first CSS framework that lets you build custom designs directly in your HTML using small, reusable classes. It helps you create responsive and modern layouts quickly without writing custom CSS.

React

React is a JavaScript library for building interactive user interfaces, especially single page applications. It uses reusable components and a virtual DOM to efficiently update and render the right parts of your UI when data changes.

PHP

PHP is a server-side scripting language used to build dynamic websites and web applications. It runs on the server and can interact with databases to generate HTML content for the browser.

Laravel

Laravel is a PHP framework for building modern web applications with clean, organized code. It includes built-in tools for routing, authentication, and database management to simplify development.

VSCode

Visual Studio Code (VS Code) is a free, lightweight code editor developed by Microsoft. It supports many programming languages and features like extensions, debugging, and Git integration.

GitHub

GitHub is a web-based platform for storing and managing code using Git version control. It allows developers to collaborate, track changes, and host projects publicly or privately.

Git

Git is a version control system that tracks changes in code and helps developers collaborate. It allows you to manage different versions of a project and merge updates from multiple contributors.
